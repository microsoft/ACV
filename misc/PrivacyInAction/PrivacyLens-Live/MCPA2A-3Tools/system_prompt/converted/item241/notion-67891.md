# Assistantship Position Workflow

Workflow for assistantship positions:  
- Application deadline set for March 1st.  
- HR screens applications by March 15th.  
- Interviews scheduled from March 20th to April 5th.  
- Hiring committee meets to make decisions by April 10th.  
- Successful candidates notified by April 15th.  
- Onboarding begins on May 1st.
